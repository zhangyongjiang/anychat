AnyChat for iOS SDK是客户端开发套件，共用AnyChat for Windows SDK（或AnyChat for Linux SDK）的服务器。

可下载AnyChat for Windows SDK，里面包含Windows版本的服务器程序：
	http://www.anychat.cn/download.html?ProName=Windows

可下载AnyChat for Linux SDK，里面包含Linux版本的服务器程序：
	http://www.anychat.cn/download.html?ProName=Linux
